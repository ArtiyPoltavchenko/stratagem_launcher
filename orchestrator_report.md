## Project Summary

Mobile PWA remote control for Helldivers 2 stratagems.
Phone → HTTP → Python/Flask server → pynput → keystrokes in game.

**Stack:** Python 3.10+, Flask, pynput, Vanilla JS/HTML/CSS (no npm).
**Environments:** WSL2 for development/tests, Windows for running the server (pynput requires native Windows).

---

## Architecture

```
Phone (Chrome PWA)
  └─ HTTP POST /api/execute {id}
       └─ Flask server (Windows, .venv_win)
            └─ keypress.py → pynput → Ctrl + WASD keys → Helldivers 2
```

**Key files:**
| File | Role |
|------|------|
| `server/app.py` | Flask app factory, all routes |
| `server/keypress.py` | pynput key simulation (VK codes, thread-safe lock) |
| `server/stratagems.py` | JSON loader + validator |
| `server/config.py` | Config dataclass (delays, host, port) |
| `data/stratagems.json` | Single source of truth — 59 stratagems, 6 categories |
| `web/index.html` | PWA shell |
| `web/app.js` | Frontend logic |
| `web/style.css` | Dark Helldivers theme |

## Running the Project

**Windows (server):**
```powershell
python -m venv .venv_win
.venv_win\Scripts\activate
pip install -r requirements.txt
scripts\start_server_wifi.bat   # WiFi mode with QR code
```

**WSL (tests/dev):**
```fish
python3 -m venv .venv
source .venv/bin/activate.fish
pip install -r requirements-dev.txt
pytest tests/   # 43 tests, all green
```

**Phone:** open `http://<PC-IP>:5000` in Chrome → Add to Home Screen.
